# Machine Learning and Data Science Assignments

This repository contains three assignments completed for the **ENCS5341 - Machine Learning and Data Science** course using **Python**.

## Assignments Overview

### Assignment 1: Data Preprocessing & Exploratory Data Analysis (EDA)
This assignment focuses on data cleaning, preprocessing, handling missing values and outliers, feature scaling, and exploratory data analysis on a customer churn dataset.

**Topics covered:**
- Data loading and inspection
- Missing data handling
- Outlier detection
- Feature scaling
- Univariate and bivariate analysis
- Correlation analysis
- Data visualization
- Insight extraction from data

---

### Assignment 2: Regression and Logistic Regression
This assignment is divided into two parts:
- **Non-linear Regression:** using ridge regression and RBF basis functions
- **Logistic Regression:** for customer churn prediction using linear and polynomial decision boundaries

**Topics covered:**
- Ridge regression
- Regularization
- RBF basis functions
- Logistic regression
- Polynomial features
- Accuracy, Precision, Recall
- ROC curve and AUC
- Model selection and generalization analysis 

---

### Assignment 3: Machine Learning Models on Travel-Destination Dataset
This assignment focuses on selecting a learning task, preprocessing the dataset, building a baseline model, developing additional machine learning models, tuning hyperparameters, and analyzing model performance.

**Topics covered:**
- Task definition
- Data preprocessing and EDA
- Nearest neighbor baseline
- Model building and evaluation
- Hyperparameter tuning
- Error analysis
- Final report writing
